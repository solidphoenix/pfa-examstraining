# PFA Examenstraining

Eine iPhone-kompatible Lernwebsite für die Pflegefachassistenz-Ausbildung.

## Inhalt

- Prüfungsmodus
- Falltrainer
- Lernkarten
- Normwerte
- Themenmatrix
- Dark Mode
- lokale Speicherung im Browser

## Nutzung

Die Website besteht aus einer einzelnen `index.html` und kann direkt im Browser geöffnet werden.

## GitHub Pages aktivieren

1. Repository auf GitHub öffnen.
2. **Settings** → **Pages** öffnen.
3. Bei **Build and deployment** als Source **Deploy from a branch** wählen.
4. Branch `main` und Ordner `/root` auswählen.
5. Speichern.

Danach ist die Website nach kurzer Zeit über die GitHub-Pages-Adresse erreichbar.

## Dateien

- `index.html` – komplette Website
- `.nojekyll` – verhindert Jekyll-Verarbeitung bei GitHub Pages
- `README.md` – kurze Beschreibung und Anleitung
